### 🛡️ AI-Powered Phishing Detection Engine with Explainable AI (XAI)
> **Python · Node.js · Express.js · MongoDB · JWT**

A full-stack cybersecurity platform that detects phishing emails and malicious URLs using interpretable machine learning, with built-in explainable AI alerts for every decision.

- 🧠 **Multinomial Naive Bayes** email classifier trained on 160 labelled emails — 94% true positive rate, 6% false positive rate
- 🌲 **Five-tree weighted Random Forest** URL analyser covering domain structure, brand impersonation, URL shorteners, punycode, and character statistics
- 🔍 **Explainable AI layer** surfacing top phishing/safe tokens and per-tree feature votes directly from model internals
- ⚡ Sub-50ms average API response time with batch support for 10 emails / 20 URLs per request
- 🔐 Production-grade security: JWT authentication, bcrypt hashing, Helmet.js headers, rate limiting

### ☁️ Autonomous Cloud SIEM — AI-Driven Intrusion Detection & SOC Dashboard
> **Python · FastAPI · scikit-learn · React · AWS CloudTrail**

A production-grade cloud security monitoring system that ingests logs, detects anomalies using ML, classifies threats using MITRE ATT&CK, and presents everything on a live SOC dashboard.

- 📥 **Log ingestion pipeline** processing 200+ AWS CloudTrail events per cycle with feature extraction
- 🤖 **Isolation Forest ML** anomaly detection — same algorithm class used in Splunk and Microsoft Sentinel
- 🎯 **Rule-based attack classification** engine mapping events to MITRE ATT&CK categories: privilege escalation, credential brute force, data exfiltration, insider threat
- 📊 **Live SOC dashboard** with real-time alert feed, ML confidence scoring, severity trend chart, search/filter, and CSV export
- 🔄 FastAPI backend with paginated `/alerts`, `/ingest`, `/stats`, and `/health` endpoints
